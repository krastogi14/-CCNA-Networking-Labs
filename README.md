# 🌐 CCNA Networking Labs

## Overview

This repository contains Cisco Packet Tracer labs covering fundamental networking concepts, including routing, switching, network services, network management, and basic security.

The labs were created as part of hands-on practice to strengthen practical networking and Cisco IOS configuration skills.

---

## Lab Categories

### Routing
- Static Routing
- Dynamic Routing
- OSPF

### Switching
- VTP
- EtherChannel
- PortFast
- Port Security

### Network Services
- DHCP, DNS & HTTP Server Configuration
- Telnet & SSH
- RADIUS Server

### Network Management
- CDP & LLDP

### Security
- Access Control Lists (ACL)
